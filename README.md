🐶 Практична робота: Реалізація успадкування в Java
📌 Опис роботи

У даній практичній роботі продемонстровано використання механізму наслідування (inheritance) в мові програмування Java.

Було створено базовий клас Animal, який містить спільні характеристики та поведінку для всіх тварин, та клас Dog, який наслідується від нього і розширює функціональність.

🧬 Реалізація наслідування
🔹 Базовий клас Animal

Містить спільні для всіх тварин:

Поля:

name — ім'я тварини

age — вік

weight — вага

Методи:

eat() — тварина їсть

sleep() — тварина спить

move() — тварина рухається

makeSound() — тварина видає звук

🔹 Клас Dog

Клас Dog наслідується від Animal та додає власні характеристики.

Додаткові поля:

breed — порода

isTrained — чи дресирована

Методи:

bark() — гавкає

fetch() — приносить предмет

перевизначений makeSound()
